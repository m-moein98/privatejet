Welcome to PrivateJet's documentation!
======================================
* PrivateJet is a fast(high-performance) and easy to use python backend framework for building REST applications.

* The key features of PrivateJet are:
   * **Fast**: is built on top of the ASGI specification and uses the uvicorn server.
   * **Easy to use**: designed to be easy to use and easy to get started with.
   * **REST**: is designed to build REST applications.
   * **Open source**: an open source project and free to use.

.. seealso:: 
   In case you are interested to contribute, this is the `PrivateJet Github:
   <https://github.com/m-moein98/privatejet>`_.
